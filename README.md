# NovaMind AI Landing Website

Simple starter project for an AI startup marketing site.

## Project structure

- `src/index.html` — main page markup
- `src/styles.css` — responsive styling
- `src/app.js` — menu and form behavior
- `src/images/` — logo and hero illustration assets

## Run locally

```bash
cd src
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
